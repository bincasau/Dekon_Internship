# TypeScript Fundamentals

## Structure

- `src/01-types.ts`: basic data types
- `src/02-generics.ts`: generics
- `src/03-utility-types.ts`: utility types

## Requirements

- Node.js
- TypeScript

## Installation

```bash
npm install
```

## Run

- npx tsx src/01-types.ts
- npx tsx src/02-generics.ts
- npx tsx src/03-utility-types.ts
